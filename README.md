# ig-nx.github.io

Sitio web personal desplegado con GitHub Pages. Este repositorio funciona como una página de presentación para mostrar información general, tecnologías utilizadas y enlaces relevantes.

## Descripción

Este proyecto consiste en una página web estática desarrollada con HTML, CSS y JavaScript. Está pensada como una plantilla base para proyectos personales, portafolios o demostraciones rápidas, sin necesidad de backend.

## Características

- Diseño limpio y responsive  
- Estructura clara por secciones (objetivo, tecnologías, estado, contribución)  
- Fácil de modificar y escalar  
- Despliegue automático con GitHub Pages  
- Sin dependencias externas obligatorias  

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages

## Despliegue con GitHub Pages (paso a paso)

Este proyecto utiliza el sistema de despliegue automático de GitHub Pages. El proceso seguido fue el siguiente:

1. Se creó un repositorio en GitHub con el nombre exacto:
   ig-nx.github.io  
   Este formato permite que el sitio funcione como página principal del usuario.

2. Se añadió el archivo `index.html`, que actúa como punto de entrada del sitio web.

3. Se subió el contenido al repositorio mediante commits en la rama `main`.

4. En la configuración del repositorio:
   - Se accedió a **Settings**
   - Se ingresó a la sección **Pages**
   - En “Source” se seleccionó:
     - Deploy from a branch
   - Se configuró:
     - Branch: main
     - Folder: / (root)

5. GitHub Pages quedó activado y el despliegue se realizó automáticamente.

6. El sitio quedó accesible en:
   https://ig-nx.github.io

## Gestión del proyecto

Este repositorio está configurado como sitio principal de GitHub Pages, lo que implica:

- El nombre del repositorio coincide con el usuario (`usuario.github.io`)
- El archivo `index.html` es la página principal
- Cada cambio en la rama `main` actualiza automáticamente el sitio
- No se requiere configuración de servidor ni hosting adicional
