# ig-nx.github.io

Sitio web personal desplegado con GitHub Pages. Este proyecto sirve como una página de presentación para mostrar información general, tecnologías utilizadas y enlaces relevantes.

## Descripción

Este repositorio contiene una página estática desarrollada con HTML, CSS y JavaScript. Está diseñada como una plantilla simple para proyectos personales o demostraciones rápidas.

## Características

* Diseño limpio y responsive
* Secciones organizadas (objetivo, tecnologías, estado, contribución)
* Fácil de modificar y extender
* Compatible con GitHub Pages

## Tecnologías

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Vista previa

El sitio está disponible en:
https://ig-nx.github.io/

## Instalación y uso

Clonar el repositorio:

```
git clone https://github.com/ig-nx/ig-nx.github.io.git
cd ig-nx.github.io
```

Abrir el archivo `index.html` en el navegador.

## Despliegue

El proyecto está desplegado automáticamente mediante GitHub Pages usando la rama `main`.

Para adaptar este proyecto:
